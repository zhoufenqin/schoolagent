# 上海学区房决策助手 · 幼升小

> 🌐 在线访问：**https://zhoufenqin.github.io/schoolagent/**

面向**买房家长**的上海学区房（幼升小）决策工具，覆盖 **徐汇、黄浦、闵行、青浦、杨浦** 五个区。

## ⚠️ 重要声明
- 本项目由 **AI 分析整理**，仅供参考，**不保证数据准确性、完整性或时效性**。
- 政策、对口划片、学校梯队、房价等信息**逐年变化**，最终请以**当年区教育局官方招生细则**、官方对口划片范围、以及实地核实为准。
- 本工具**不构成任何购房、入学或投资建议**。

## 功能
- **区域 / 学校对比**：5 区评分排名、落户时机测算、中考升学数据、供给侧（缩班）提示。
- **学位 / 落户自查**：输入家庭情况，测算 2029 入学对口风险 + 官方核验清单。
- **学区 / 小区**：
  - 好学校名录（公办/民办分标、梯队、对口/摇号）——点公办学校可展开**对口小区 + 地图**。
  - 公办对口小区（学区房）——徐汇/黄浦/闵行已落地 **2026 官方对口居委/小区**。

## 数据来源（标注可信度）
- 政策类：上海市教委、各区教育局、上海市义务教育入学报名系统、教育考试院（2024–2026 官方）。
- 对口划片：各区《2026 公办小学招生划片范围》官方原文（详见 `suishenban/`）。
- 房价 / 升学：公开认知整理或权威媒体转引，标注"估计·待核实"。
